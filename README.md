## Project Setup

1. **Clone the repository**:
   ```sh
   https://github.com/Nitin-Singh18/product_app.git
   ```

2. **Navigate to the project directory**:
    ```sh
    cd product
    ```

3. **Install dependencies**:
    ```sh
    flutter pub get
   ```

4. **Run code generator**:

   ```sh
   dart run build_runner build
   ```

5. **Run the project:**:
    ```sh 
   flutter run
    ```
